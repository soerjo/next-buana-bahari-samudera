# Nextjs Web Buana Bahari Samudera

Web Buana Bahari Samudera Nextjs Project.

## Getting Started

> Run the development server:

```bash
npm install
npm run dev
```

> Run the production server:

```bash
npm install
npm run build
npm run start
```

## Tech

- React
- Nextjs
- Tailwind
- framer-motion
